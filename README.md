# Blog-node
搭建个人网站-node

**不使用任何框架** ，Node.js搭建个人网站，后面看情况慢慢加上一些常见功能
###已完成的功能：
- 基本的静态服务(html,css,js，img)
- 处理ajax请求,回传json格式数据
- 构建类似express和koa风格的项目，流式处理中间件
- ejs模板引擎
- 用mime处理Content-Type
- 重定向到首页
- 白名单cookie实现登录
- 动态路由
- MongoDB存储数据，mongoose
- pm2部署
- element组件开发

###待完成的功能：
- 图片上传
- 账号密码登录
- 密码加密
- 错误和日志系统
- Redis
- 图片资源存储第三方服务
- 微信登录
- MongoDB分布式系统
- 断点续传
- 不中断地重启Node服务
- nginx
